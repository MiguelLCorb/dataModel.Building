<!-- 40-RequiredProperties -->  
Cette entité contient une description harmonisée d'un bâtiment. Cette entité est associée aux segments verticaux des maisons intelligentes, des villes intelligentes, de l'industrie et des applications IdO connexes. Ce modèle de données a été partiellement développé en coopération avec les opérateurs de téléphonie mobile et la [GSMA] (https://www.gsma.com/iot/iot-big-data/). Par rapport au modèle de données de la GSMA, les changements suivants ont été introduits la référence à `BuildingType` est supprimée, car l'attribut `BuildingType` comparé à l'attribut `category` n'apporte pas d'informations significatives. L'attribut `category` est obligatoire. `openingHours` est introduit suivant le modèle de données schema.org pour permettre une granularité fine sur les heures d'ouverture des bâtiments. GSMA l'a soutenu en tant que texte libre dans l'attribut `notes` (supprimé également). `refSubscriptionService` n'est pas supporté, puisque le modèle `SubscriptionService` n'est pas supporté actuellement.
